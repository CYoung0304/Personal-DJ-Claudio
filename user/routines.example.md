# 我的日常作息（示例）
#
# Claudio 会根据当前时间从这份作息推断"现在适合放什么"。
# 改成你自己的就行。

## 早晨 07:00 - 09:00
- 起床、洗漱、早餐
- 心情：慢慢清醒，需要温柔的音乐
- 适合：轻快的 indie pop，清新民谣

## 上午工作 10:00 - 12:00
- 深度工作，需要专注
- 心情：专注，不想被打扰
- 适合：lo-fi, ambient, post-rock

## 午餐 12:00 - 14:00
- 休息、午饭
- 心情：放松
- 适合：bossa nova、爵士

## 下午工作 14:00 - 18:00
- 继续工作，下午可能有些疲惫
- 适合：chillhop, trip-hop, 中等能量电子

## 晚间 18:00 - 22:00
- 下班，晚餐，个人时间
- 适合：华语独立、爵士、古典

## 深夜 22:00+
- 准备睡觉或安静阅读
- 适合：ambient, 极低能量
