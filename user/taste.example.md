# 我的音乐品味（示例）
#
# 把你喜欢的风格、艺术家、避雷的东西都写在这里。
# Claudio 在选歌和说话时会读这份资料，越具体越能挑到你喜欢的歌。

## 喜欢的风格
- Lo-fi / Chillhop
- Jazz（尤其是 bossa nova）
- 华语独立 / 民谣
- Post-rock
- 电子 ambient

## 喜欢的艺术家
- 陈奕迅
- Norah Jones
- Nujabes

## 不喜欢的
- 重金属
- 过于喧闹的 EDM

## 能量偏好
- 工作时：低能量，不要有太多歌词
- 运动时：高能量，节奏感强
- 晚上放松：舒缓，爵士或古典
- 早上起床：轻快，给人好心情
